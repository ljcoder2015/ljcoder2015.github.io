1. HTTP是什么？

HTTP是一个客户端终端（用户）和服务器端（网站）请求和应答的标准，通常，由HTTP客户端发起一个请求，创建一个到服务器指定端口（默认是80端口）的TCP连接。HTTP服务器则在那个端口监听客户端的请求。一旦收到请求，服务器会向客户端返回一个状态，比如”HTTP/1.1 200 OK”，以及返回的内容，如请求的文件、错误消息、或者其它信息。

tabBarItem设置title会被ViewController.title的标题所替换的问题

发表于 2016-03-17 | 分类于 iOS开发 | 阅读次数

在开发是我们可能会遇到这样的问题，当我们在ViewController里面设置导航标题是，一般会采用下面的语法：

self.title = @“NavigationTitle”;

虽然达到了我们的目的，但是把tabBarItem也替换掉了，可是我们明明设置的是其它标题

[self.tabBarItem initWithTitle:@”Home” image:[UIImage imageNamed:@”iconHome.png”] tag:0];

说明tabBarItem的title会优先采用ViewController的title，解决办法就是直接设置navigationItem的标题

// self.title = @”NavigationTitle”
self.navigationItem.title = @”NavigationTitle”;

在平时开发项目的过程中，我们一般都会使用git或者svn来进行版本控制，公司的话，可能会买一个服务器来搭建一个svn，如果是我们自己个人使用，买个服务器就不划算了，还好现在网盘一大把，大容量又免费，今天就教大家怎么使用微云来搭建一个svn服务器。

思路

大体思路就是使用SmartSvn创建本地仓库，然后使用微云同步盘把存放本地仓库的文件夹同步到微云上，然后再其它电脑上同步下来，再使用SmartSvn check out仓库就可以了。所以要使用到的工具就是SmartSvn和微云同步盘。
